# base58

Base58 utility to encode integers as strings and vice versa.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  base58:
    github: russ/base58
```


## Usage


```crystal
require "base58"

Base58.decode("6hENdZ")
Base58.encode(3470417529)
```
